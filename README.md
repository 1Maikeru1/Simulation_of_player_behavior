# Симуляция поведения игрока

При запуске приложения на экране генерируется заданное количество кругов. Игрок взаимодействует с кругами, нажимая на них левой кнопкой мыши. При нажатии на круг:
- Объект исчезает.
- Появляется в новом случайном месте в границах экрана.
- Фиксируются данные о взаимодействии.
Сбор данных осуществляется по следующим параметрам:
- игрок — идентификатор пользователя.
- сессия — уникальный номер сессии.
- время — момент взаимодействия.
- действие — тип действия (клик).
- объект — идентификатор объекта.
- позиция X, Y — координаты объекта на экране.
- время реакции — интервал между появлением объекта и кликом.
- баллы — начисленные очки за действие.
